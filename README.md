# Anarti Implementation Plan

A Next.js presentation deck for the Anarti Therapeutics implementation plan. Features interactive slides covering strategic AI integration, lead nurture engines, content authority systems, and project timeline.

## Features

- 10 interactive slides with full navigation
- Dark theme optimized for presentations
- Client-side rendering with React
- Fully responsive layout
- Previous/Next navigation and slide indicators

## Getting Started

### Development

```bash
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
npm start
```

## Deployment

This app is ready to deploy on Vercel. Connect your GitHub repository and deploy with zero configuration.

## Tech Stack

- **Next.js 16** - React framework with App Router
- **React 19** - UI library
- **CSS-in-JS** - Inline styles for full control
